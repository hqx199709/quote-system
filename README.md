# 自动报价系统使用说明

这是一个快捷、专业的自动报价工具，旨在帮助您快速生成一致且精美的报价单。

## 文件结构
- `index.html`: 系统主界面。
- `app.js`: 核心逻辑代码。
- `products.js`: **产品清单配置文件（修改此处来更新您的产品信息）**。

## 如何修改产品清单？
打开 `products.js`，按照以下格式修改或添加项目：

```javascript
{
    name: "分类名称",
    items: [
        { 
            id: "唯一ID", 
            name: "产品名称", 
            price: 1000, 
            unit: "单位", 
            description: "详细描述" 
        }
    ]
}
```

## 功能特点
1. **即选即得**：点击左侧配置，右侧自动更新价格。
2. **精美模板**：预设了商务风格的报价单样式。
3. **PDF 导出**：点击“导出 PDF”按钮，利用浏览器的打印功能即可保存为高质量 PDF 格式。

## 快速开始
只需在浏览器中打开 `index.html` 即可开始使用。
